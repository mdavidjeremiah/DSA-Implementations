Linked List

Nodes connected by pointers. Dynamic size, O(1) insertion at head, sequential traversal. Singly & doubly linked.

Binary Search Tree

Hierarchical node structure. Left < root < right invariant. O(log n) search, insert & delete on balanced trees.

Vertices connected by edges. Directed & undirected. Adjacency list representation. BFS and DFS traversal built in.
