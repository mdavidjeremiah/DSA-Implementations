Queue ADT

A FIFO (First-In, First-Out) abstract data type. Elements are added at the rear and removed from the front — just like a real-world queue. Used in BFS traversal, task scheduling and IO buffering.

// ADT Implementation (JS)
class Queue { constructor() { this.items = []; } // Add to rear — O(1) enqueue(element) { this.items.push(element); } // Remove from front — O(n) array dequeue() { if (this.isEmpty()) throw new Error("Queue underflow"); return this.items.shift(); } // View front element — O(1) front() { if (this.isEmpty()) throw new Error("Queue is empty"); return this.items[0]; } isEmpty() { return this.items.length === 0; } size() { return this.items.length; } }
