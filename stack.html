Stack ADT

A LIFO (Last-In, First-Out) abstract data type. Elements are added and removed from the same end (top). Push values in, pop them off — the last item pushed is always the first out.

// ADT Implementation (JS)
class Stack { constructor() { this.items = []; } // Add element to top — O(1) push(element) { this.items.push(element); } // Remove & return top — O(1) pop() { if (this.isEmpty()) throw new Error("Stack underflow"); return this.items.pop(); } // View top without removing — O(1) peek() { if (this.isEmpty()) throw new Error("Stack is empty"); return this.items[this.items.length - 1]; } isEmpty() { return this.items.length === 0; } size() { return this.items.length; } }
